# esp8266 twi arduino nano

- twi line locked down and never pullup again when slave too slow in respond
- solve by increase setting clock strech limit

```c
Wire.begin(D2, D1);
auto ms = 7;
// default is 230us
Wire.setClockStretchLimit(ms * 1000); // us
```
